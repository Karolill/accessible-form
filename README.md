# accessible-form

A React application demonstrating accessible vs inaccessible form design.

## Prerequisites
- Node.js (v18 or higher)
- npm or pnpm

## Getting Started

### Install dependencies
```bash
npm install
# or
pnpm install
```

### Run development server
```bash
npm run dev
# or
pnpm dev
```

### Build for production
```bash
npm run build
# or
pnpm build
```

### Preview production build
```bash
npm run preview
# or
pnpm preview
```

## Features
- Side-by-side comparison of accessible vs inaccessible form design
- Accessible form showcases: labels, fieldset/legend, ARIA attributes, validation, autocomplete
- Inaccessible form demonstrates common accessibility mistakes
